# Aprovest
Projeto de desenvolvimento de uma aplicação que utiliza gamificação para ajudar os estudantes do 3º ano do ensino médio no Brasil a estudadarem para o vestibular.

# Tecnologias Utilizadas
- HTML
- CSS
- JAVASCRIPT
- MYSQL
- PHP
