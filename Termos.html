Termos de privacidade

1. Informações que Coletamos

Para proporcionar a melhor experiência de estudos, coletamos:

  • Dados de cadastro: Nome, e-mail, senha, série/ano e vestibulares de interesse
  • Dados de estudos Área de interesse, respostas, dúvidas
  • Dados de do sistema Pontos e conquistas

2. Como Utilizamos suas Informações

Seus dados nos ajudam a:

  • Analisar o perfil dos alunos para melhorar nossos conteúdos
  • Recomendar matérias e conteúdos que precisam de mais atenção
  • Criar relatórios de progresso e metas de aprendizado
  • Enviar comunicados sobre quando os vestibulares escolhidos irão começar

3. Compartilhamento de Dados

Não vendemos seus dados pessoais. Compartilhamos informações apenas:

  • Com provedores de serviços essenciais para operação da plataforma
  • Para cumprir obrigações legais ou solicitações judiciais
  • De forma agregada e anônima para fins estatísticos e de pesquisa educacional

4. Seus Direitos

Você tem o direito de:

  • Acessar e corrigir suas informações a qualquer momento
  • Solicitar uma cópia de todos os dados que possuímos sobre você
  • Solicitar a exclusão de sua conta e dados pessoais

5. Segurança de Dados

Implementamos medidas robustas para proteger suas informações:

  • Seus dados são criptografados quando armazenados
  • Acesso restrito apenas a pessoal autorizado
  • Monitoramento contínuo contra ameaças à segurança
  • Backups regulares para evitar perda de informações

6. Menores de Idade

Nossa plataforma é voltada para estudantes, inclusive menores:

  • Coletamos dados de menores com consentimento dos pais/responsáveis
  • Não coletamos dados de crianças menores de 13 anos sem verificação parental
  • Pais/responsáveis podem solicitar a exclusão de dados de menores a qualquer momento
